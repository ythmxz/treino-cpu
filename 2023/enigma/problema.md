# Enigma (CPU)

Limite de tempo: 1 segundo

## Problema:

As crianças são muito inventivas em seus jogos. Principalmente quando se trata da linguagem falada ou escrita. Na linguagem falada um exemplo é a língua do “P”, mas o mais intrigante é na linguagem escrita. As crianças sabem que os pais (zelosos) monitoram tudo que elas escrevem ou recebem por escrito. Então já aprendem, desde cedo, “criptografar” suas mensagens. Um exemplo é o jogo: ZENIT-POLAR. Esta é uma criptografia por substituição: a letra Z é trocada nos textos pela letra P e vice-versa e assim as letras sucessivas do jogo. A seguir um exemplo de uma frase em ZENIT-POLAR: “Osri citri olagmiraci lie o ziti quinquot um not” Podemos traduzir facilmente conhecendo o código: ”Esta carta enigmatica nao e para qualquer um ler”. Só que para ser mais rápido foi pedido a você um algoritmo que dada uma frase a traduza. Observe que a frase pode estar no português para traduzir para o ZENIT-POLAR, ou em ZENIT-POLAR, para traduzir para o português.

## Entrada:

A entrada apresenta várias linhas cada linha contém um texto não acentuado com caracteres ASCII padrão. Cada linha tem tamanho máximo de 100 caracteres. A entrada termina com o final do arquivo.

## Saída:

Para cada linha de entrada deve haver uma linha na saída com o texto traduzido respeitando-se maiúsculas e minúsculas.

| Exemplo de entrada:                              | Saída do exemplo de entrada:                     |
| ------------------------------------------------ | ------------------------------------------------ |
| Osri citri olagmiraci lie o ziti quinquot um not | Esta carta enigmatica nao e para qualquer um ler |
| Quero cifrar um texto usando ZENIT-POLAR         | Quote caftit um roxre usilde POLAR-ZENIT         |
